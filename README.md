Bike Sales Dashboard – Excel Data Analysis Project
Project Overview
This project presents an interactive Excel dashboard created to analyze customer demographics and purchasing behavior related to bike sales.
The dashboard transforms raw customer data into meaningful insights using Pivot Tables, charts, and slicers.
The objective of this project is to demonstrate data analysis and data visualization skills using Microsoft Excel by identifying factors that 
influence bike purchase decisions.

**Dataset Information**
The dataset contains 1026 customer records with demographic and behavioral attributes.
~Key Variables
~Marital Status
~Gender
~Income
~Number of Children
~Education Level
~Occupation
~Home Ownership
~Number of Cars
~Commute Distance
~Region
~Age
~Purchased Bike (Target Variable)

**Tools & Skills Used**
 ~Microsoft Excel
 ~Data Cleaning and Preparation
 ~Pivot Tables and Pivot Charts
 ~Data Visualization
 ~Dashboard Design

**Dashboard Features**
The Excel dashboard provides visual insights including:
~Average Income Comparison (Bike Buyers vs Non-Buyers)
~Customer Age Distribution
~Bike Purchases by Region
~Analysis of Commute Distance and Purchase Behavior
~Interactive slicers for filtering data by customer attributes
~These visualizations help identify patterns in customer purchasing behavior.

**Key Insights**
Some insights from the analysis include:
~Customers with higher income levels are more likely to purchase bikes.
~Commute distance influences bike purchasing decisions.
~Certain age groups show higher interest in bike purchases.
~Regional differences highlight varying customer demand.
These findings can support businesses in targeted marketing and customer segmentation strategies.
